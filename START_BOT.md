# Запуск бота

## Быстрый старт

1. Убедись что Python 3.10+ установлен
2. Установи зависимости: `pip install -r requirements.txt`
3. Заполни `.env` файл (скопируй из `.env.example`)
4. Запусти: `python bot.py`

## Команды

- `/start` — приветствие
- `/weather` — выбор города
- `/help` — справка

Бот работает автономно, никакой веб-деплой не нужен!
